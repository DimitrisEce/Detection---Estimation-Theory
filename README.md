# Detection---Estimation-Theory
Small project on Detection and Estimation Theory using Matlab.

The goal of the assignment is to construct and apply a Weiner Filter on multichannel EEG data for post-processing. The provided dataset and the project instructions are not included as they are not the property of the owner of this repository.
